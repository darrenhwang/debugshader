# 模糊转场效果使用说明

## 一、效果概述

这是一个模拟模糊效果的转场，支持自定义模糊半径和边缘发光效果。适用于场景切换、技能释放等需要模糊效果的场景。

## 二、使用方法

### 1. 基础设置
1. 在 Cocos Creator 中创建新材质
2. 选择 `blur_transition` 效果
3. 将材质应用到目标精灵上

### 2. 参数说明

#### 基础参数
- `progress`：转场进度
  - 范围：0.0 - 1.0
  - 默认值：0.0
  - 说明：控制转场进度，0表示开始，1表示完成

- `blurRadius`：模糊半径
  - 范围：1.0 - 20.0
  - 默认值：5.0
  - 说明：控制模糊的程度

- `edgeWidth`：边缘宽度
  - 范围：0.0 - 0.5
  - 默认值：0.1
  - 说明：控制发光边缘的宽度

- `edgeColor`：边缘颜色
  - 类型：颜色
  - 默认值：[1.0, 1.0, 1.0]
  - 说明：控制边缘发光的颜色

## 三、效果预设

### 1. 轻微模糊
```json
{
  "progress": 0.5,
  "blurRadius": 3.0,
  "edgeWidth": 0.1,
  "edgeColor": [1.0, 1.0, 1.0]
}
```

### 2. 中等模糊
```json
{
  "progress": 0.5,
  "blurRadius": 5.0,
  "edgeWidth": 0.15,
  "edgeColor": [0.8, 0.8, 1.0]
}
```

### 3. 强烈模糊
```json
{
  "progress": 0.5,
  "blurRadius": 10.0,
  "edgeWidth": 0.2,
  "edgeColor": [1.0, 0.8, 0.0]
}
```

## 四、使用场景

### 1. 场景切换
- 使用轻微模糊效果
- 调整进度实现平滑过渡
- 可以添加边缘发光增加视觉效果

### 2. 技能释放
- 使用中等模糊效果
- 调整边缘颜色匹配技能效果
- 控制转场速度创造打击感

### 3. 特殊效果
- 使用强烈模糊效果
- 调整模糊半径创造梦幻感
- 结合其他效果增强表现力

## 五、注意事项

### 1. 性能优化
- 避免使用过大的模糊半径
- 控制转场进度变化速度
- 适当调整效果参数

### 2. 常见问题
- 如果效果不明显，检查模糊半径
- 如果边缘太锐利，增加边缘宽度
- 如果性能下降，降低模糊半径

### 3. 最佳实践
- 根据场景需求选择合适的预设
- 微调参数获得最佳效果
- 可以结合其他效果增强表现力
